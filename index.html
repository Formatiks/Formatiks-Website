Frequently asked questions

What is Formatiks?

Formatiks is a modular platform for structured content and repeatable workflows — connect templates, automations, and your own extensions.

What are modules?

Modules are small, composable units — templates, scripts, or integrations — that extend Formatiks without bloating the core app.

Where can I find modules?

Browse the Module library for curated and community-built modules you can install in one tap.

Is Formatiks free?

Yes — the core experience is free. Advanced team features may ship later with clear, optional pricing.

Which platforms are supported?

Today we focus on iOS and macOS for the native clients. Web tools are on the roadmap.
